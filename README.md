# React 写一个A站

## 使用说明

注意下使用的npm还是yarn，下面吧npm和yarn的步骤都说明一下

### 安装

```
    npm install 
    yarn
```

### 运行

```
    npm start 
    yarn start  
```

### 打包

```
    npm build
    yarn build 
```

## 项目中使用架构

- react
- reactRouter
- antd
